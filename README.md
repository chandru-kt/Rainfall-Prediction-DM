# Rainfall-Prediction-DM

## Run in Rstudio

### The objective of this project is to develop a predictive model for rainfall prediction using data mining   techniques. The project aims to forecast the amount of rainfall in a specific location, based on historical data such as temperature, humidity, wind speed, and other meteorological factors.
